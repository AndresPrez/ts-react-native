This directory ("navigations") contains all code that handles navigation in the app. It will basically have all the navigation logic
between the different Screen components under "screens".

"React Navigation" will be used for handling navigation in the app
https://reactnative.dev/docs/navigation

- @react-navigation/native
- @react-navigation/stack

The following ones are recommended by React Navigation
- react-native-reanimated
- react-native-gesture-handler
- react-native-screens
- react-native-safe-area-context
- @react-native-community/masked-view
